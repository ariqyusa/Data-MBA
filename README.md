# Data-MBA
Tugas Python for Data Science: Permainan Tebak Angka

Menu:
- Main
- Daftar
- High Score
- Keluar

Main:
- Generate random number from 0-20
- Input: 
  - Registered username and password
  - float/int (dg statement "Masukkan tebakan Anda")
- Print:
    - "Saya memikirkan sebuah angka dari 0-20, silahkan tebak angka yang saya pikirkan, input harus berupa angka"
    - Hint: Terlalu besar atau terlalu kecil
    - Score saat ini
- Penalties:
  - Salah angka: -10
  - Salah input type (bukan float/int): -20

Daftar:
- Ask player for username and password, export to txt file
- Password tidak visible saat input

High Score:
- Save the highest score of a user, export to txt file

Keluar: 
- Kembali ke menu
